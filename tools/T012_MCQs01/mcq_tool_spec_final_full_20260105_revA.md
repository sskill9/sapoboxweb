# Multiple Choice Questions ツール 仕様書（確定版）

## 1. 目的
本ツールは、新人研修などで用いる「選択式テスト（MCQ）」を、ブラウザだけで運用できる形で提供する。
サーバ保存は行わず、オフライン運用を前提とする。

## 2. 想定利用者
- SV：設問作成、採点、結果出力
- 受講者：受験、結果確認

## 3. 利用環境
- Edge / Chrome
- ローカルHTML（ZIP展開運用）

## 4. 基本方針
- testpack.js を読み込んで受験
- 結果はTXTでダウンロード

## 5. 採点・合否
- 合格率は小数第1位まで
- 合否は「合格／不合格」

## 6. 採点キー
- 固定キー
- 画面表示・保存・ログ出力は禁止

## 7. 結果TXT
### 7.1 ファイル名
社員番号_氏名_テスト名_合否_YYYYMMDD-HHMMSS.txt

### 7.2 内容
- テスト情報
- 受講者情報
- 正解率・合否
- 合格時のみ設問別 正解／不正解

※ 正しい選択肢や解説文は出力しない
